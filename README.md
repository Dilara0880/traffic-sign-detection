# Веб-приложение для детекции дорожных знаков 

Это веб-приложение для автоматической детекции дорожных знаков с помощью дообученной модели YOLOv8. Пользователи могут загружать изображения с дорожными знаками, и приложение автоматически определяет, какие знаки присутствуют на картинке.

## Функционал
- **Загрузка изображений**: Пользователи могут загружать изображения в формате JPEG для обработки.
- **Детекция объектов**: Приложение использует модель YOLOv8 для детекции объектов на загруженных изображениях.
- **Отображение результатов**: Обнаруженные объекты отображаются с ограничивающими рамками на оригинальном изображении, а также с соответствующими метками классов.
- **Адаптивный дизайн**: Приложение спроектировано так, чтобы быть удобным и адаптивным, обеспечивая плавный опыт работы на различных устройствах.

**Стек: Django, YOLOv8, SQLite, HTML/CSS** 

## Установка

1. **Клонировать репозиторий**:

   ```bash
   git clone https://github.com/Dilara0880/traffic-sign-detection.git
   cd detection
   ```
2. **Настроить окружение**:
   ```
   python -m venv .venv
   source .venv/bin/activate  # На Windows используйте `.venv\Scripts\activate`
   ```
3. **Установить зависимости**:
   ```
   pip install -r requirements.txt
   ```
4. **Настроить базу данных**:
   ```
   python manage.py migrate
   ```
5. **Создать файл конфигураций**:
   Создайте файл **config.yaml** со следующими параметрами:
   ```
   secret_key: "YOUR-SECRET-KEY"
   debug: true
   allowed_hosts:
   - localhost
   - 127.0.0.1
   ```
6. **Запустить локальный сервер**:
   ```
   python manage.py runserver
   ```

## Дообучение модели
Ноутбук [YOLOv8_train.ipynb](https://github.com/Dilara0880/traffic-sign-detection/blob/main/detection/YOLOv8_train.ipynb) содержит подробную информацию по дообучению модели yolov8n.pt. _Использованы 3530 изображений для обучения, 801 изображение для валидации и 638 для тестирования._ 
Структура директории датасета:
```
yolo_train/
├── data/
│   ├── train/                  # Папка для обучения
│   │   ├── images/             # Папка с изображениями
│   │   ├── labels/             # Папка с метками
│   ├── val/                    # Папка для валидации
│   │   ├── images/             
│   │   ├── labels/               
│   ├── test/                   # Папка для тестирования
│   │   ├── images/             
│   │   ├── labels/   
├── models/
│   ├── yolov8n.pt              # Предобученная модель
│   ├── custom_model.pt         # Обученная модель после тренировки
├── config.yaml                 # Конфигурационный файл для обучения
└── requirements.txt           # Зависимости проекта
```
